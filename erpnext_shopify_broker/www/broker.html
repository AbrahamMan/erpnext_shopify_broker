ERPNext - Shopify Connector

ERPNext Connector is OAuth based bridge between Shopify and ERPNext. This connector helps in,

  • Sync Orders from Shopify to ERPNext

  • Sync Products and Cutomers in both way

ERPNext helps you take full control of your business. It is as powerful as it is easy to use.

There are two steps to get connected with ERPNext via Shopify APP

  1. Connect via Public App
  2. Connect via Private App

Connect via Public App

Steps To Follow:

  1. Install ERPNext-Shopify Connector from ERPNext Installer

  2. Get ERPNext Connector from Shopify App Store

  3. Fillup ERPNext-Shopify Connector form and click on submit

  4. Permission page with install button will popup. It will ask permissions to access,

    • Modify Products, variants and collections

    • Modify Customer details and customer groups

    • Modify Orders, transactions and fulfillments

  5. Setup Shopify Settings

  6. You can sync data, by clicking Sync Shopify button from Shopify Settings or via scheduler which will sync data after an hour.

Connect via Private App

Steps To Follow:

  1. Install ERPNext-Shopify Connector from ERPNext Installer

  2. From Shopify admin desk you can create private app, APP > Private App

  3. From Private App, pickup API key and password and place them inside ERPNext Shopify Settings

  4. Setup Shopify Settings

  5. You can sync data, by clicking Sync Shopify button from Shopify Settings or via scheduler which will sync data after an hour.
